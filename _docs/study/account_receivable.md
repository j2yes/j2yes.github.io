# 매출채권 관리 실무

> 채권 사고의 사전 예방관리 방법 파악, 
> 어음수표와 담보관리 방법 이해, 
> 채권보전과 집행권원 획득 방법, 
> 강제집행 실무의 개념과 절차 


## 비재무적 요인에 대한 신용조사
- 거래처 사업력, 사업장 경영자 조사
- 거래처 조직 분위기, 매출, 손익추이 
- 거래처 부동산 등기부 등본, 법인 등기부 등록
- 거래처 자동차 등록원부 주민등록초본
- 부도나 위기 징후 파악

### 거래처 사업력, 사업장, 경영자 조사
when : 새로운 거래시작, 소문 안좋을 때, 연체, 거래 조건 변경 요청, 정기적으로 조사
- 사업력
  - 사업력 7년 이상이면 양호
  - 자본, 매출액 증가 추세면 양호
- 사업장
  - 회사 분위기
  - 자가소유 사업장
  - 생산, 판매시설 재투자
  - 사업장 환경(깨끗)
- 경영자
  - 소유자와 경영자 명의가 같은지
  - 동종업계 경력이 긴경우 좋음 
  - 주위 평판 (회사 and 경영자)

### 주요 거래처와 종업원에 대한 조사

  - 주요 거래선이 어디인가? 
  - 거래선이 다수인가?
  - 주요 거래선이 우량 업체 인가?
  - 주요 거래선의 결제조건
  - 거래처 수주잔액 많으면 좋음
-종업원
  - 종업원의 사기
  - 종업원의 근속연수

### 부동산 등기부등본, 법인 등기부등록
- 부동산 등기부등본 : 주소지알면 열람 발급 가능
```
 표제부 (물건에 관한 사항 기재)
 갑구 : 소유권에 관한 사항 (가압류, 가처분, 경매, 가등기, 환매등기 등)
 을구 : 전세권, 저당권 등
```
  - 사업장, 대표자 거주지가 대표자의 본인 명의 인가?
  - 자기소유라도 권리관계(갑구)가 복잡한지 확인
  - 가압류, 근저당권에 제3금융권, 개인이 있는 경우 문제
- 법인 등기부등록(사업장과 대표자 소재지, 거주지 알 수 있음) : 회사명 법인등록번호로 열람 발급 가능 
```
 설립일 (사업이력 확인)
 납입자본금 (매출액의 5~15%)
 이사(임원) 변동
 기타사항 (기업회생,파산절차, 가처분)
```

### 자동차 등록원부, 주민등록 초본 열람
- 자동차 등록원부 : 자동차등록번호, 소유자로 확인
  - 권리관계와 권리자 확인(신용 파악)
  - 세금 체납으로 압류 사실있는지 확인
- 대표자 주민등록 초본 : 채권 채무 관계에 있는 경우 열람 가능(법제처 국가법령센터 주민등록법 시행규칙 참고)
  - 과거 주민등록 말소 이력 확인
  - 주민등록 자주 변경하는지 확인 
  - 최근 주민등록 변경 사실
  - 채무자 단독세대는 문제일 확률 높음

## 거래처의 재무평가 방법
- 재무제표 중 재무상태표와 손익계산서 파악
- 거래처의 재무제표를 열람하는 방법 파악
- 주요 재무비율 계산법을 통해 재무비율을 계산
- 신용정보 회사의 신용정보 활용

### 재무상태표와 손익계산서를 보고 신용 평가하기
- 재무제표 구하기
  - 재무제표는 신용정보회사에서 알 수 있음
  - 금융감독원 전자공시 시스템 (자산총계 100억이상, 코스닥 상장업체)
  - 거래처에 요청
- 재무상태표 : 일점 시점의 재무상태를 나타냄
  - 자산 (부채 + 자본) : 유동자산(1년 기준 현금화 가능), 비유동자산(1년 이후)
  - 부채 (= 타인자본) : 유동부채, 비유동부체
  - 자본 (= 자기자본)
- 손익계산서 : 이익계산표, 일정 기간 기업의 손익현황을 나타냄
  - 매출액 - 매출원가 = 매출 총이익 (생산 효율성)
  - 매출 총이익 - 판매비아 관리비 = 영업이익 (영업 효율성)
  - 영업 이익 - 영업 외 손익(이자비용, 이자수익, 외환차익) = 법인세 차감 전 순이익
  - 법인세 차감 전 순이익 - 법인세 = 당기 순이익 
- 신용문제
  - 이익이 나는데, 단기 차입금 증가추세면 문제가 있을 수 있음(분식, 흑자도산)
  - 자기자본 + 비유동 부채 < 비유동 자산 문제 , 단기차입금도 무리하게 비유동 자산에 투자했을 수 있음
  - 이자비용/(차입금+사채) 동종업계보다 큰 거래처
  - 매출액 대비 매입채무규모가 클 경우
  - 현금, 현금성 자산 적거나 감소 (종업원수 * 500만원 안되면 문제)
- 분신 가능성
  - 재무구성비율 변동이 심한 경우
  - 이익 소폭씩 증가
  - 선수금, 선급금 과다한 경우 (비용을 선급금 등으로 만들었을 수 있음)
  - 매출채권, 재고자산 구성비율 20% 이상인 경우

### 거래처 재무비율 산정방법
- 유동성 비율 : 단기채무를 갚을 수 있는 능력(단기 지불능력) , 채권자 입장에서 제일 중요
  - 유동비율 = 유동자산 / 유동부채 * 100
  - 당좌비율 = 당좌자산 / 유동부채 * 100
- 안정성 비율 : 장기적인 지불능력
  - 부채비율 = 부채 / 자기자본 * 100
  - 비유동 장기적합률 = 비유동자산 / (자기자본+비유동부채) * 100 : 일반적으로 70% 수준
- 수익성 비율 : 수익성 판단 기준
  - 매출액 영업이익률 = 영업이익/ 매출액 * 100
  - 매출액 순 이익률 = 순이익 /매출액 * 100
  - 총자본 순 이익률 = 순이익 /총자본 * 100
- 활동성 비율 : 소유 자산이 얼마나 효과적으로 이용되고 있는지 측정하는 비율율
- 성장성 비율 : 기업이 적정하게 성장하는지 
  - 매출액 증가율
  - 총자본 증가율
- 생산성 비율 : 경영합리화 나타내는 비율
  - 부가가치율
  - 설비투자효율

### 지수법에 의한 거래처 재무평가 방법
평가항목 | 가중치 | 재무비율 | 산업평균 | 관계비율 | 조정 비율 | 평점
- 한국은행 홈페이지 경제통계 시스템, 동종업계 산업평균비율 조회
- 높을 수록 좋은 재무비율  (= 거래처 비율 / 동종업계평균비율)
- 낮을 수록 좋은 재무비율 (=동종업계평균비율 / 거래처 비율)
- 조정 관계비 = 관계비율 * 0.7
- 평점 = 가중치 * 조정관계비율 (평점이 70보다 크면 양호)

### 신용정보회사의 신용정보를 활용하는 방법

## 채권회수 방법
- 채무이행 심리 파악
- work sheet 활용법
- 채권회수 활동 실무 적용
- 정상적 회수 x => 간접강제하는 방법

### 채무이행 심리를 활용한 채권회수기법
- 채권회수 마인드
  - 채권회수 완료해야 회사에 이익
  - 부실채권 역승수효과
- 채권은 1~2개월 내에 회수하도록 해야 함 (시간이 지날수록 채무이행의지 희박해짐)
- 변제 약정일 (1차 약정일 회수 안되면 2차 약정일 15일 이내 다시 약속) : 3~5차례 연기 전까지 회수 가능해짐
- 면담
  - 먼저 변제청구
  - 다툼 X
  - Win-Win하는 협상
  - 끈기있고 집요하게
  - 논리적으로 설득
  - 지불 계획서(각서)를 가지고 만남
- 유의할 점
  - 통화시간은 짧게
  - 감정을 상하게 하지 않음
- 채무자 유형별 채권회수 방법
  - 연체기간 짧고 지불의사 있는 경우 : 변제 약정일 약속 받고 회수활동, 연장회수는 3~5회 넘기지 않게함
  - 연체기간 짭고 지불의사 없는 경우 : 독촉장 , 법 조치
  - 연체기간 길고 지불 유예 요청 : 지불각서, 공정증서받고 어길 경우 법 조치 
  - 연체기간 길고 지불의사 없는 경우 : 법 조치
- 채권회수 절차 : 변제 기일에 변제 청구 -> 다음 변제 기일에 변제 청고 -> 간접강제(수시방문 독촉, 연대보증인 청구) -> 법 조치 -> 소제기 -> 강제 집행
- 채권회수 대상과 재산
  - 채무자가 개인 : 채무자, 보증인, 채무인수인
  - 일상가사대리(자녀학비..)는 부부연대채무
  - 채무자가 법인 : 채무자 법인, 보증인

### work sheet 활용법
채권회수를 체계적으로 하기위해 사용
- 회수계획은 변제약정일 기준으로 수립
- 변제약정일은 구두 또는 계약서에 명시
- 한달 반이 지나가면 전화가 아닌 방문

### 간접강제의 활용방법
- 일반적 간접강제
  - 악성채무자는 통화시간 관계없이 끈질길게
  - 악성채무자는 통화보다 방문(일부 회수 또는 각서받기)
  - 연대보증인이나 물상보증인에게 변제 청구
  - 최고장(독촉장)에 일정한 기간(7~15일)주어 채무이행하도록 함, 기간 내 이행하지 않으면 법 조치
  - 집행권훤 획득하면 재산명시 신청과 채무불이행자 명부등재 신청(돈이 있는데 갚지 않는 경우)
  - 채권자로서 채무자의 파산신청
 
## 독촉장 발송 / 채권 양도 / 물품회수 / 채무인수 방법
- 독촉장 발송 방법 이해, 채권 회수
- 채권 양도, 물품양수 시 주의점
- 채권양도와 물품회수를 채권회수에 활용
- 채무인수 활용

### 임의회수는 언제하는가?
- 임의 회수 방법 : 독촉장 발송, 채권양도, 물품회수, 채무인수
- 채무자 부실화 징후를 보이거나, 연체기간이 장기화 되는 경우

### 독촉장 발송방법
- 명기 사항 : 송달 주소, 변제기일(원금, 이자), 변제 기한, 법 조치할 것을 명기
- 내용증명으로 통보
- 주채무자 반응이 없으면, 연대 보증인, 담보 제공자에게 
- 내용증명 송달 방법 : 내용안에 받는사람, 보내는 사람 주소와 성명을 포함. (3통을 우체국에 제출하여 송달)
- 내용증명은 어떤 내용으로 보냈는지 증명할 수 있음
- 문서의 확정일자를 부여 
- 효력 
  - 채권의 소멸시효가 중단 (6개월 이내 법조치 해야 함)
  - 기간 경과 시 계약 해제 해지 가능
  - 의무자가 권리자에게 통지한 경우 이를 게을리 하게 되면 권리 행사 제한
- 독촉한 입증자료로 활용
- 독촉장은 한번만 보내고 반응이 없다면 법조치를 바로 해야 함

### 채권양도의 활용
- 채권양도 FLOW : 채권자 - 채무자 - 제3채무자
- 채권양도 방법 
  - 채권양도 통지 및 승낙은 확정일자 있는 문서로 함
  - 채권양도 통지는 양도인이 내용증명으로 통지해야 함
- 채권양도를 받을 때 주의점
  - 채권증서를 받아야 함 (세금계산서, 거래명세표 등)
  - 신속히 받아야 함 (중복 방지)
  - 양도 통지 및 승낙은 확정일자 있는 문서로 하여야 함
  - 채무자와 제3제무채간 상계할 수 있는 채권이 있는지 확인
- 대금의 지급에 갈음하여, 대금지급에 대신하여 라는 문구를 넣어서는 안됨
- 대금의 지급을 위하여 라고 채권양도 받아야 함
- 경우(양도인의 변제능력이 없는 경우 : 사해행위 취소 소송 대비)에 따라 대금의 지급에 갈음하여라고 채권양도 받아야 함

### 물품양수 방법
- 거래처가 부실화 되는 경우 채권을 물품으로 양도
- 신속성이 필요
- 형사적 문제가 되지 않도록 회수 (상대방에게 동의를 얻어야 함)
- 품목, 규격, 수량 명확히 하여 양수를 받아야 함
- 환가 금액, 방법, 시기 명확히 합의
- 대물변제는 (갈음하여)변제하기 때문에 => 채무자가 변제할 채무 중 금 xx원의 변제에 대신하여라고 명시해야 함

### 채무인수의 활용
- 채무를 인수인에게 이전 : 항변권도 그래도 이전됨
- 면책적 채무인수 : 채무자 , 채무인수인 중 채무인수인만 책임
  - 채권자 승낙을 얻은 경우에 한하여 가능
  - 채무자 의사에 반하여 체결하지 못함
  - 채무자에 비해 인수인의 신용 상태가 양호해야 함
- 중첩적 채무인수 : 채무자 , 채무인수인 둘 다 책임
  - 채권자 승낙없이 인수계약 가능
  - 채무자 의사와 무관하게 인수 계약 가능
- 인수계약 당사자 : 채권자, 채무자, 인수인


## 계약서 작성과 체결방법
- 계약서 작성시 주의할 내용
- 등록된 인감과 인감증명서를 받아 놓았을 때 유리한 점
- 서명날인과 기명날인
- 계약 체결과 날인 방법

### 계약서 작성시 주의할 점
- 작성 전 검토 사항
  - 작성 이유 : 당사자간 이해 증진, 법률효과, 분쟁 예방 및 최소화
  - 조건이 회사 규정에 적정한지
  - 거래 조건이 이행 가능한지 검토
- 계약서 착안사항
  - 정확(일시, 금액 등 구체적으로), 간결, 명료 : 미사여구를 달지 않음
  - 용어의 통일
  - 관련문서는 계약서에 첨부
  - 숫자, 단위, 주요조항 정확성 검토
  - 강행 법규에 위배되는지 검토
- 계약서 작성 방법
  - 계약 물품의 품명, 수량, 가격 (복잡한 경우 별첨하여 명기)  
  - 기본계약서 / 개별계약
  - 물품의 인도 : 인도장소, 상하차 장비
  - 검수와 검사 : 검수와 검수 통보 기한 명시
  - 하자보증 : 기산점(인도일, 검수일), 하자보증 범위를 명확히 정해야 함(소모품 등 제외)
  - 대금지급 : 가능한 60일이내, 연체이자율을 정함
    - 연체이자율 정하지 않은 경우 : 민사채권 5%, 상사채권(사업자) 6% 
  - 소유권 유보 : 물품의 금액 전부 결제된 경우 매수인에게 소유권 이전
  - 규격의 변경 : 납기, 단가에 영향을 미치면 조정 가능하도록
  - 상표사용 및 지적재산권 불침해 보장
  - 계약의 해제 : 계약 해제 사유 명기 (구첵적으로)
### 등록된 인감과 일반인감 / 서명날인과 기명날인
- 등록된 인감 (인감도장, 법인인감) : 법률효과 있음, 인감도장이 날인되고 인감증명서 첨부되어 있다면 진정 성립된 것으로 추정(도장 주인이 입증)
- 일반인감 : 일반도장, 법률효과 있음, 증거력이 확실하지 않음
- 서명날인 : 자필로 상호 성명 기재하고 날인, 법률효과 있음, 증명이 용이
- 기명날인 : 서명 포함, 프린터로 성명 표기하고 날인, 법률효과 있음, 증명이 쉽지 않음
### 계약 체결과 날인 방법
- 계약일자 기재
- 계약서 여러장인 경우 간인을 함(계약서 갈아끼지 못하도록 함)
- 계약 당사자의 주소, 상호, 성명 기재하고 날인 받음 (주민등록번호 or 법인등록번호)
- 공동대표인 경우 공동대표 모두의 날인을 받아야 함 (대법원홈페이지 법인등기부등본 확인)
- 약관에 의한 계약 체결(ex:보험) : 일방 당사자가 다수의 상대방과 일정한 형식과 내용으로 미리 마련한 계약
  - 명시, 설명의 의무
    - 약관 사본을 고객에게 교부하여야 함
    - 중요한 내용을 고객이 이해할 수 있도록 설명하여야 함
    - 규정을 위반한 경우 계약의 내용으로 주장할 수 없음
- 무효가 되는 약관조항 (법제처 `약관` 검색)
  - 고객의 해제, 해지권의 부당한 제한
  - 고객에게 불리한 소제기 금지조항
- 계약서를 정정해야 하는 경우
  - 계약물품, 규격, 금액, 납기 변경하는 경우
  - 대금지급 방법이나 지급기일 변경
- 계약서 정정 방법
  - 삭제 : 두 줄을 긋고 행의 맨 앞 여백에 `삭제 X자` 명기, 양 당사자 날인
  - 정정 : 두 줄을 긋고 두 줄 그은 부분 위에 정정할 내용 기재, `정정 X자` 명기하고 양 당사자 날인
  - 삽입 : 삽입기호 표시를 하고 냉ㅇ 삽입, `가(첨) X자` 명기하고 양 당사자 날인

## 계약관련 법률 지식
- 불공정 계약 등 계약이 무효가 되는 경우와 취소할 수 있는 계약
- 동시이행의 항변
- 계약의 해제와 해지의 차이점
- 공증

### 계약의 무효(불공정 계약)와 취소
- 무효가 되는 경우
  - 선량한 풍속 사회질서에 반하는 내용
  - 악의, 과실이 있는 진의 아닌 의사표시
  - 통정의 허위 표시 (서로 사실이 아님을 알고 한 계약)
  - 불공정, 강행법규에 반하는 계약
  - 궁박, 경솔, 무경험으로 현저히 공정성을 잃는 계약
  - 권리 의사 무능력자와의 계약
- 불공정 거래 행위
  - 공정거래법 (법제처 홈페이지에서 법령검색, 공정거래 위원회 법령검색)
  - 거래상의 지위를 부당하게 이용하여 상대방과 거래하는 행위 (우월적 지위의 남용)
- 취소할 수 있는 계약
  - 취소의사를 상대방에 통보해야 무효가 됨
  - 사기, 강박, 중요 착오가 있는 계약
  - 제한 능력자(미성년자, 금치산자 ...)와의 계약

### 동시이행의 항변, 위험부담 기한의 이익 상실
- 동시이행의 항변권 : 이행을 거절할 수 있는 항변권 (상대방이 서로 맺은 계약을 완전히 이행하고 있는 않은 경우) 
  - 효력 : 상계하지 못함, 이행지체 책임이 없음
- 위험부담 : 체무자가 자기 채무를 면하고, 채권자에 대한 권리도 잃음 
  - 채권자가 지체하여 수령하는 경우 발생한 피해는 채권자가 의무를 져야 함
- 기한의 이익 상실 
  - 채무자가 담보 제공의무를 이행하지 않는 경우
  - 가압류, 압류, 파산, 기업회생절차, 어음 수표 부도
  - 재산처분, 대표자 변경으로 채무면탈 기도

### 계약의 해제와 해지 방법
- 계약 이행 전의 계약해제
  - 계약 이행 : 계약금 이외의 돈, 물품을 지급(일부 포함)
  - 매수한 사람 (계약금), 매도한 사람 (계약금 * 2)
- 계약의 해제와 해지권
  - 약정 해제
  - 법정 해제권 : 이행지체, 이행불능, 불완전이행, 수령지체
  - 법정 해지권 : 이행지체, 이행불능, 불완전이행
- 계약 해제 방법
  - 계약 이행에 대한 유예기간 (보통 14일)을 주고 통지
  - 상대방이 이행하지 않을 의사를 표시한 경우와 정기행위(특정기간이내에 이뤄져야하는 계약)는 유예기간 없이 바로 해제, 해지 가능
- 해제 : 단발적 거래
  - 소급적으로 무효
- 해지 : 계속적 거래
  - 장래에 대해서 무효

### 공증의 활용
- 공증 : 법률관계 공적으로 증명
- 효력 : 분쟁 방지, 집행력을 갖음
- 사서증서의 인증 : 계약 당사자가 작성한 계약서, 합의서, 보증서 등 서류상에 한 서명날인이 본인의 의사에 의한 것이라는 것을 확인하고 기재한 것 (확실한 증거력)
- 공정증서의 작성 : 채무자의 강제집행을 인락한 취지의 문구가 기재된 공정증서 (증거력 + 집행권원)
- 확정일자 : 공증기관이나 관공서에서 찍힌 일자 (작성된 일자에 문서의 존재 증명)
- 공증기관 및 구비서류
  - 기관 : 공증인, 법무법인, 검찰청 등
  - 서류 : 채권증서, 채권자 신분증, 도장, 공증용 위임장, 채무자 인감증명서, 법인등기부등본

## 어음 및 수표 수취 시 주의사항
- 어음 수표의 피리요적 기재사항
- 발행일이나 수취인이 없는 어음 수령시 조취 방법
- 유해적 기재사항 어음 수표 받지 않는 방법
- 어음 수표 수령 시 주의사항
- 부도나기 쉬운 어음 판별, 고의부도 징후 감지

### 약속어음과 수표의 필요적 기재사항
- 어음 수표 받으면 유리한 점
  - 확실한 채권증서
    - 지급거절 시 부도처리, 은행거래 정지
    - 원인 채무오 어음채무 선택적으로 권리 행사 가능
    - 어음, 수표는 단독심의로 처리
    - 대손처리(비용으로 인정)
- 약속어음 필요적 기재사항
  - `약속어음`이라는 문언, 만기, 수취인, 발행일, 발행인 ...
- 수표 필요적 기재사항
  - `수표`라는 문언, 지급인, 발행인 ...
- 발행일과 수취인이 없는 어음 수표를 수취한 경우
  - 배서인에 대한 청구권을 행사할 수 없음
  - 보충권이 소지인에게 위임된 것으로 봄
  - 만기가 발행일 보다 빠르면 무효
- 자수어음, 타수어음
- 어음 수표의 보충 규정
  - 만기 : 일람출급으로 봄
  - 지급지 : 발행지를 지급지로 봄
  - 발행지 : 발행인 명칭의 부기한지를 발행지로 봄

### 유해적 기재사항
- 유해적 기재사항(기재되면 어음, 수표 무효)
  - 조건부 
  - 분할지급
  - 지급 한정
  - 제한
  - 지시금지는 예외 (배서하지 말라는 뜻)
- 무효가 되는 어음 수표
  - 선택기재 (200 또는 300만원)
  - 불확정 기재 (30 내지 300만원)
  - 유동적 기재 (금10돈)
  - 문자와 숫자 다른 경우 (문자우선)
  - 기명 / 날인 하나만 찍힌 경우
  - 날인 대신 무인(지장)
  - 부전지에 발행인이 기명날인한 경우

### 부도나기 쉬운 어음, 수표 구별법
- 융통어음 : 상거래 없이 발행한 어음
- 진성어음 : 상거래를 하고 교부받은 어음
- 부도나기 쉬운 융통어음
  - ex) 발행인 - 배서인(거래처) - 소지인(우리)
  - 발행인과 배서인이 거래관계가 희박하다면 융통어음으로 생각하면 된다.
- 딱지어음
  - 자료상 신용도 올린후 어음 발행하여 유통
- 교환어음
  - 여러 사람들과 빌려주고 빌려받는 어음
- 회전어음 : 배서인이 많은 어음
  - 어음에 배서인이 많은 경우 금융기관에서 할인받을 수 없을 정도로 발행인 & 배서인의 신용도가 낮은 경우가 많음
- 채무자로부터 고액의 약속어음을 받고 차액을 돌려달라는 경우
  - 만기일에 차액돌려주기로 해야함
  - 신용도 높은 회사의 발행 어음이더라도 어음으로 차액을 줘야함 (배서하지 못하도록 지시금지하여 발행)
- 지급기일이 일요일이나 공휴일인 경우 부도확률이 높음
- 악성루머 업체가 발행한 어음
- 발행일이 없는 어음
- 만기일 개서하여 연장한 어음
- 만기가 장기인 어음

### 고의부도의 행태
- 발행인 재력에 비하여 큰 금액의 어음
- 주문량이 급격히 증가 (덤핑 판매 후 부도 가능)

- 어음 수표의 도난, 분실 시 적절한 조치방법

- 부도 시 일반적인 조치사항
- 피사취 부도 시 조치방법

### 어음, 수표의 도난 분실 시 조치사항
- 도난 분실 : 공시최고 등 적절한 조치
- 부도 처리 : 권리보전 등
- 피사취 부도 처리 : 어음대금청구 소송 등
- 분실 시 조치 사항
  - 발행인 : 발행인 직접 사고신고
  - 소지인 : 은행에 우선 신고(효력 없음), 발행인에게 통지하여 지급은행에 서면으로 사고 신고(지급정지)
  - 경찰에 도난 분실 신고, 신문에 분실 도난 사실공고
  - 법원에 공시최고 신청 : 재권판결
- 공시최고 신청 (첨부자료 : 지급은행의 미지급증명서, 분실 도난 신고 증명서, 어음 수표의 사본이나 발행인의 발행증명서...)
  - 공시최고 신청서에 구비서류를 첨부하여 공시 최고 신청 (대법원 - 전자민원신청 - 절차안내 - 신청)
  - 공시 최고 기간 내에 소지자가 법원에 신고하면 분실자와 신고자중에서 정당한 권리자 판결 (6:4 or 5:5 등)
  - 기간 이내(보통 3개월) 소지인으로부터 신고가 없으면 제권판결
  - 절차 : 공시최고 신청 -> 공고 -> (3개월) -> 공시최고 기일에 신청인 출석하여 진술 -> 제권판결 -> 공고
- 제권판결
  - 증권 무효 판결 : 어음 수표의 권리를 무효화 시키는 증권 무표 판결
  - 제권판결 후 선의취득을 할 수 없음
  - 발행이나 배서인의 면책력 없어짐
  - 분실한 사람은 수표의 원본이 없어도 수표의 권리행사가 가능함

### 어음 수표의 부도
- 은행도 어음 : 은행에서 발행한 어음 용지를 사용하여 발행한 약속어음 (지급기일에 결제를 못하면 부도처리 됨) 
- 환어음 vs 약속어음(일본,한국만 존재)
- 개인어음 : 발행인이 임의로 만든 어음 (지급기일에 결제를 못해도 부도처리 되지 않음), 개인어음은 공증을 받아서 집행권한을 얻어야 함
- 당좌수표(지급기일이 없음) : 지급하지 못하면 부도처리
  - 연수표, 선일자수표 : 발행일은 미래로 하여 발행 (그 기일이 아니더라도 은행에서 바로 수령가능)
- 어음 수표의 부도 발생 시점
  - 당좌거래 은행 계좌에 자금 부족 : 당좌차월한도를 사용하면 마이너스통장처럼 사용가능
  - 당좌거래 은행과 거래 중단
  - 형식불비가 되어 부도 : 인감이 다름, 기명 및 날인 누락, 당좌인감(은행인감)이 아닌 경우, 약정용지가 아닌 경우, 배서불비(수취인 -> 피배서인 연속)
- 예금부족에 의한 부도처리 과정
  - 교환의뢰 -> 교환회수(만기전날 오후7시) -> 대금입금(만기일 오후2시) -> 연장(만기일 오후4시) -> 1차부도 -> 2차부도(익일)
- 부도 후 상황
  - 거래은행과 당좌거래 해지
  - 국내 은행과 만2년간 거래정지
  - 국내의 가맹점과 여신 및 당좌거래 불가
  - 배서인에게 소구권(청구권)행사 가능

### 부도 시 일반적 조치사항
- 예금부족으로 인한 부도 시 조치사항
  - 발행인과 보증인에 직접청구
  - 배서인에게 청구
  - 최대한 발행인, 보증인, 배서인과 협상 
  - 발행인, 보증인, 배서인이 재산(부동산 등 등기등록, 유체동산, 채권) 조사하여 재산에 가압류
  - 발행인, 보증인, 배서인에게 지급명령 신청이나 반환청구소송 제기
  - 집행권원을 얻어 주채무자, 보증인, 배서인의 재산에 강제 집행
- 권리보전 조치
  - 거래은행에서 부도확인을 받기 (대손채권처리)
  - 바로 이전 배서인에게 4일 이내에 통지, 통지받은 배서인은 바로 앞 배서인에게 2일이내 통지
- 기타 부도 시 조치사항
  - 형식불비 부도 시 : 내용 보정하여 지급제시 기일 내 제시하여 지급 청구 
  - 제시기간 경과 : 발행인 보증인에게 직접 청구
  - 인감 상이 부도 : 은행 거래 인감을 날인하여 지급제시 기일(만기일 다다음날) 내 지급제시

### 피사취 보도 시 대처 방법 
- 피사취 부도(계약 불이행하는 경우 발행인이 부도 처리) 의 절차 
  1. 지급제시 전에 발행인이 지급은행에 사고신고서 제출
  2. 사고 신고 담보금 처리를 위한 약정 체결
  3. 지급제시 전 사고신고 담보금을 별단예금에 예탁
- 피사취 부도 vs 일반부도
  - 피사취 부도 은행 거래정리 처분 없음
  - 은행과 여신 당좌거래 가능
  - 부정 수표 단속법상 제재를 받지 않음
- 피사취 부도 대체 방법
  - 발행인과 배서인을 찾아가서 지급 설득
  - 발행인과 배서인에게 어음대금 청구 소송하여 어음금액을 회수
  - 6개월이 지나면 사고신고담보금을 발행인이 찾아갈 수 있기때문에 6개월 내 소송(지급 은행에 소제기 증명원 제출)
  - 은행에서 사고신고담보금을 상계한 경우 은행에 지급 요청하여 처리

## 전자어음 및 전자결제
- 전자어음의 장점
- 기업구매카드 활용 주의할 점
- 전자방시기 매출채권담보 대출 활용
- 기업구매 자금대출제도 활용

### 전자어음에 대한 이해
- 전자결제가 필요한 이유
  - 어음 수표의 분실, 도난 등으로 인한 피해 방지
  - 사회적 폐해 완하
  - B/B거래 증가 추세
- 전자어음 : 전자적으로 발행, 관리, 금융기관에 등록
  - 2014.04 ~ : 총 자산 10억 이상 법인은 전자어음 발행 의무
  - 발행, 배서, 보증이 전자적으로 처리 (어음 할인 가능)
- 전자어음 발행인과 수취인
  - 발행인 : 거래은힝에 당좌예금계약 체결 -> 공인인증서 발급 및 본인확인 -> 인터넷뱅킹에서 공인인증서로 전자어음 발행
  - 수취인 : 공인인증서 발급 -> 거래은행에 가입 및 전자어음 이용약정 체결 -> 거래은행 인터넷 뱅킹(또는 unote에서 확인 가능)에서 발행여부 확인
- 전자어음의 장점
  - 물류 비용 절감
  - 분실 도난 방지
  - 거래 투명성 강화
  - 발행한도가 제한되므로 부도발생 비율 낮아짐

### 기업구매 카드
- 어음제도 문제점 개선을 위해 도입 : 자동으로 납품대금 지급
  1. 구매기업이 은행과 계약 체결 : 은행이 납품대금 지급 업무대행
  2. 은행이 납품업체와 가맹 체결
  3. 구매업체가 은행에 납품내역 전송
  4. 은행이 납품업체에 수수료 공제 후 대금지급
  5. 대금결제일에 구매업체가 은행에 지급
- 장점 
  - 구매기업 : 카드발급 연회비 면제, 구매카드 통한 결제 시 법인세 공제, 어음지급에 따른 비용 절감
  - 판매기업 : 외상매출채권 즉시 회수에 따른 자금운용유리, 대금분할수령 가능, 외상매출채권 회수 관리 비용절감
- 구매업체가 은행에 대금지급을 제대로 하지 못하는 경우 납품업체가 지급해야하는 경우가 있을 수 있음

### 전자방식 외상매출채권담보부 대출
- 구매기업이 은행과 납품대금 지급업무 대행 계약 체결, 판매기업이 외상매출채권 담보도 대출 받아 납품대금 조기 상환받음
- 판매기업 신청 7일 이내 
- 외담대 장점
  - 구매기업 : 신용공여한도 , 지급기일 관리 용이
  - 판매기업 : 조기에 현금화 가능
- 외담대 대출업무 
  1. 판매기업이 구매기업에 납품
  2. 판매기업이 은행에 대출신청
  3. 구매기업이 은행에 납품내역 전송
  4. 판매기업 대출
  5. 구매기업이 은행에 납품대금 지급
- 외담대 상환은 구매기업이 하지만 구매기업이 상환하지 못하는 경우 판매기업이 부담해야 함
- 전자채권 : 외담대와 거의 유사, 금융결제원이 주관하여 운영

### 기업구매 자금대출제도
- 구매기업이 물품대금을 어음으로 지급하는 대신 은행에서 대출받아 현금으로 지급
- 장점
  - 은행 : 대출 이자 확보
  - 납품 업체 : 현금으로 납품대금수령
  - 구매업체 : 저리이자

## 근저당권의 설정과 사후관리 방법
- 근저당권 설정 전 체크사항
- 부동산 물건 권리분석
- 근저당권 설정방법
- 근저당권이 사후관리 방법

### 근저당권 설정방법
- 근저당권의 권리
  - 우선변제권 확보
  - 임의경매를 신청할 수 있음
  - 다른 채권자가 신청하여 경매가 진행될 경우도 배당받을 수 있음
- 저당권 
  - 확정 채무 / 채권액 기준 
- 근저당권 
  - 미 확정 채무 / 채권 최고액
- 근저당권 설정방법
  - 근저당권 설정계약 -> 등기등록(부동산, 자동차 등)
  - 나대지에 설정할 때 지상권도 동시에 설정해야 함
  - 적정 설정 금액 : (감정평가액 - 선순위 설정액 - 임차보증금 - 임대하지 않은방 * 3400 / 2700 / 2000 / 1700) = 담보여력 * 120 ~ 150%
  - (확정일자받은 임차보증금 담보보다 우선변제권을 갖음)

### 법정지상권 / 권리분석
- 법정지상권 : 토지와 그 지상건물이 각각 소유자가 다른 경우 건물소유자를 위하여 지상권이 설정된 것
  - 무허가, 미등기 건물도 성립
  - 건축 중인 경우도 성립
- 부동산 등기부 선 순위 설정여부
  - 표제부 : 물건에 대항 사항 (주소, 면적, 구조, 용도...)
  - 갑구 : 소유권 사항 (가압류, 가등기, 가처분...)
  - 을구 : 소유권 이외 사항 (저당권, 전세권...)
  - 같은 구에서 등기 순서번호로 우열이 가려짐
  - 갑구와 을구 사이는 접속일자와 접수번호로 우열을 가림
- 주택임대차 보호법
  - 대항 요건 (입주 및 전입신고)
  - 확정일자 (근저당권과 일자비교)
  - 최우선 변제권 (임차보증금이 서울 1억, 수도권8천만, 광역시 6천만, 기타 5천만 이하인 경우 임차주택 가액의 1/2범위 안에서 일정금액을 우선 변제 받을 수 있음)
- 상가건물 임대차 보호법
  - 대상 : 임차보증금 + 차임(얼세) * 100 (서울 4억, 수도권 3억, 광역시 2.4억...)
  - 대항 요건을 갖춘 임차인 : 저당권, 가압류, 가등기담보, 양도담보 등이 없는 상가건물에 입주하고 사업자 등록 신고
  - 확정일자를 갖춘 임차인 : 대항요건과 확정일자를 갖추면 우선변제권 확보
  - 소액임차인의 최우선 변제권 
- 조세채권과의 관계
  - 법정기일 전 근저당권 설정된 경우 : 근저당권 우선
  - 법정기일 후 근저당권 설정된 경우 : 국세, 지방세 우선
  - 경매되는 물건에 부과되는 세금은 세금이 우선
- 임금채권과의 관계
  - 임금과 세금 : 임금이 우선 (근저당권보다 우선인 세금은 세금이 우선)
  - 임금과 근저당권 : 근저당권이 우선
  - 임금 채권의 최우선 변제권 : 최종 3월분 임금, 최종 3년간 퇴직금, 재해보상금은 최우선변제권을 갖음
- 권리 우선 순위
  - 1순위 : 경매집행비용
  - 2순위 : 최우선 변제권 (소액임차인, 최우선임금)
  - 3순위 : 경매되는 물건에 부과되는 세금(당해세)
  - 4순위 : 저당권 설정 전 국세, 지방세
  - 5순위 : 저당권, 확정일자 있는 임대차 보증금
  - 6순위 : 일반 임금
  - 7순위 : 저당권보다 후순위의 국세, 지방세
  - 8순위 : 공과금
  - 9순위 : 일반 채권자 채권(남는 돈 비율로 나눔)

### 근저당 물건 변동 시 조치할 사항
- 물상대위권 : 근저당 물건에 대한 보상금 지급 전 압류해야 함
- 건물증축, 개축, 멸실
  - 부동산 등기부의 표제부가 달라진 경우 : 담보를 새로 설정해야 함
  - 근저당권 관련 조치 : 담보 추가 설정 하거나 법인을 채무자로 추가하는 채무자 변경등기

### 채무자 변경 시 사후관리 사항

- 법인사업자 법인등록번호 변경

## 질권, 양도담보, 소유권 유보의 활용
- 동산질권의 설정 및 집행방법
- 권리질권의 설정 및 집행방법
- 양도담보의 설정 및 집행방법
- 소유권 유보의 활용방법

### 동산질권의 설정 및 집행방법
- 동산(등릭등록되는 동산 이외의 동산)을 담보로 받을 때는 양도담보와 질권으로 받는다.
- 채무자의 채권 양도담보와 질권으로 받는다.
- 물적 담보의 종류
  - 유치권 : ex)건물을 점유할 수 있는 권리 (건물 공사비 등...)
  - 저당권 : 등기등록이 되는 재산(부동산, 자동차...)에 설정 
  - 질권 : 유체동산, 채권
  - 담보권 : (2012.06~) 동산이나 채권도 등기를 하여 담보 설정 가능
  - 양도담보 
  - 가등기담보 
  - 소유권유보
- 동산질권의 설정
  - 채권자와 질권설정자간 질권설정계약 체결 -> 담보 목적물을 인도, 변제가 완료시 까지 유치 -> 변제되지 않는 경우 우선변제 받을 수 있는 권리
  - 동산질권 설정 가능 물건 : 동산(등록이 되는 재산 외의 모든 동산), 카메라, 귀금속, 명품 등
- 동산질권의 효력과 집행
  - 담보되는 채권의 범위 : 원본, 이자, 손해배상금 등
  - 목적물 : 양도된 물건의 전부
  - 우선변제권 확보 
  - 실행 : 경매절차에 의해 담보물건 환가하고 우선변제 / 담보목적물로 직접 청구
  - 소멸 : 변제완료 / 담보 목적물을 질권 설정자에게 반환한 때 (점유하지 않으면 효력이 없어짐)

### 권리질권의 설정 및 집행방법
- 권리질권 설정 목적물 : 채권, 주식, 예금 등 양도가능한 재산권
- 설정방법 : 채권자와 질권 설정자간 질권설정 계약 체결 -> 채권증서 수령 -> 제3채무자로부터 승락 또는 질권자가 질권설정을 통지(승낙이나 통지는 확정일자 있는 문서로 해야 제3자에게 대항할 수 있음)
- 권리질권의 집행 : 제3채무자에게 양도 청구 / 압류, 양도, 매각 명령 신청을 하여 강제집행

### 양도담보의 설정 및 집행방법
- 채권자와 질권설정자가 계약을 체결하고 공정증서(집행력이 인정된 공정증서, 강제집행 권한을 얻기 위함)를 받음
- 점유개정(인도는 서류상으로 받고 실제점유는 상대방이 하고 있음)
- 효력 : 물건의 점유와 사용
  - 범위 : 제한이 없음
  - 담보물건을 선의 취득한 제3자와의 관계 : 선의(양도담보설정여부를 알고 있었는지)라면 제3자가 소유권을 취득하게 됨
  - 담보물건에 대한 일반채권자의 압류 : 제3자 이의의 소
- 양도담보 실행방법
  - 실행 : 담보물건 상태파악 -> 담보물 인도청구 -> (승낙)담보물 처분 or (거절)강제집행
  - 처분정산형 : 제3자에게 매각하여 환가 정산
  - 취득정산형 : 담보물건 가격을 평가 정산 후 담보물건을 담보권자가 취득 
- 동산의 양도담보와 질권의 차이
  - 양도담보는 인도는 서류상으로 실제 점유는 상대방이 해도 됨(점유개정), 질권은 점유하고 있어야 함

### 소유권 유보의 활용방법
- 소유권은 인도조건에 따라 달라짐 
- 소유권 유보 : 물품대금결제가 다 완료되기 전까지 물품의 소유권은 매도자에게 있음을 명기(=물품대금 완납시점에 매수자에게 소유권 이전되는 것으로 명기)
- 대급 완납 후 물품의 손실이나 파손의 매수인이 책임
- 소유권 유보의 효력
  - 완납전 소유권은 매도인에게 있음

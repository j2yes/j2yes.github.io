# 매출채권 관리 실무

> 채권 사고의 사전 예방관리 방법 파악, 
> 어음수표와 담보관리 방법 이해, 
> 채권보전과 집행권원 획득 방법, 
> 강제집행 실무의 개념과 절차 


## 비재무적 요인에 대한 신용조사
- 거래처 사업력, 사업장 경영자 조사
- 거래처 조직 분위기, 매출, 손익추이 
- 거래처 부동산 등기부 등본, 법인 등기부 등록
- 거래처 자동차 등록원부 주민등록초본
- 부도나 위기 징후 파악

### 거래처 사업력, 사업장, 경영자 조사
when : 새로운 거래시작, 소문 안좋을 때, 연체, 거래 조건 변경 요청, 정기적으로 조사
- 사업력
  - 사업력 7년 이상이면 양호
  - 자본, 매출액 증가 추세면 양호
- 사업장
  - 회사 분위기
  - 자가소유 사업장
  - 생산, 판매시설 재투자
  - 사업장 환경(깨끗)
- 경영자
  - 소유자와 경영자 명의가 같은지
  - 동종업계 경력이 긴경우 좋음 
  - 주위 평판 (회사 and 경영자)

### 주요 거래처와 종업원에 대한 조사
- 거래처
  - 주요 거래선이 어디인가? 
  - 거래선이 다수인가?
  - 주요 거래선이 우량 업체 인가?
  - 주요 거래선의 결제조건
  - 거래처 수주잔액 많으면 좋음
-종업원
  - 종업원의 사기
  - 종업원의 근속연수

### 부동산 등기부등본, 법인 등기부등록
- 부동산 등기부등본 : 주소지알면 열람 발급 가능
```
 표제부 (물건에 관한 사항 기재)
 갑구 : 소유권에 관한 사항 (가압류, 가처분, 경매, 가등기, 환매등기 등)
 을구 : 전세권, 저당권 등
```
  - 사업장, 대표자 거주지가 대표자의 본인 명의 인가?
  - 자기소유라도 권리관계(갑구)가 복잡한지 확인
  - 가압류, 근저당권에 제3금융권, 개인이 있는 경우 문제
- 법인 등기부등록(사업장과 대표자 소재지, 거주지 알 수 있음) : 회사명 법인등록번호로 열람 발급 가능 
```
 설립일 (사업이력 확인)
 납입자본금 (매출액의 5~15%)
 이사(임원) 변동
 기타사항 (기업회생,파산절차, 가처분)
```

### 자동차 등록원부, 주민등록 초본 열람
- 자동차 등록원부 : 자동차등록번호, 소유자로 확인
  - 권리관계와 권리자 확인(신용 파악)
  - 세금 체납으로 압류 사실있는지 확인
- 대표자 주민등록 초본 : 채권 채무 관계에 있는 경우 열람 가능(법제처 국가법령센터 주민등록법 시행규칙 참고)
  - 과거 주민등록 말소 이력 확인
  - 주민등록 자주 변경하는지 확인 
  - 최근 주민등록 변경 사실
  - 채무자 단독세대는 문제일 확률 높음

## 거래처의 재무평가 방법
- 재무제표 중 재무상태표와 손익계산서 파악
- 거래처의 재무제표를 열람하는 방법 파악
- 주요 재무비율 계산법을 통해 재무비율을 계산
- 신용정보 회사의 신용정보 활용

### 재무상태표와 손익계산서를 보고 신용 평가하기
- 재무제표 구하기
  - 재무제표는 신용정보회사에서 알 수 있음
  - 금융감독원 전자공시 시스템 (자산총계 100억이상, 코스닥 상장업체)
  - 거래처에 요청
- 재무상태표 : 일점 시점의 재무상태를 나타냄
  - 자산 (부채 + 자본) : 유동자산(1년 기준 현금화 가능), 비유동자산(1년 이후)
  - 부채 (= 타인자본) : 유동부채, 비유동부체
  - 자본 (= 자기자본)
- 손익계산서 : 이익계산표, 일정 기간 기업의 손익현황을 나타냄
  - 매출액 - 매출원가 = 매출 총이익 (생산 효율성)
  - 매출 총이익 - 판매비아 관리비 = 영업이익 (영업 효율성)
  - 영업 이익 - 영업 외 손익(이자비용, 이자수익, 외환차익) = 법인세 차감 전 순이익
  - 법인세 차감 전 순이익 - 법인세 = 당기 순이익 
- 신용문제
  - 이익이 나는데, 단기 차입금 증가추세면 문제가 있을 수 있음(분식, 흑자도산)
  - 자기자본 + 비유동 부채 < 비유동 자산 문제 , 단기차입금도 무리하게 비유동 자산에 투자했을 수 있음
  - 이자비용/(차입금+사채) 동종업계보다 큰 거래처
  - 매출액 대비 매입채무규모가 클 경우
  - 현금, 현금성 자산 적거나 감소 (종업원수 * 500만원 안되면 문제)
- 분신 가능성
  - 재무구성비율 변동이 심한 경우
  - 이익 소폭씩 증가
  - 선수금, 선급금 과다한 경우 (비용을 선급금 등으로 만들었을 수 있음)
  - 매출채권, 재고자산 구성비율 20% 이상인 경우

### 거래처 재무비율 산정방법
- 유동성 비율 : 단기채무를 갚을 수 있는 능력(단기 지불능력) , 채권자 입장에서 제일 중요
  - 유동비율 = 유동자산 / 유동부채 * 100
  - 당좌비율 = 당좌자산 / 유동부채 * 100
- 안정성 비율 : 장기적인 지불능력
  - 부채비율 = 부채 / 자기자본 * 100
  - 비유동 장기적합률 = 비유동자산 / (자기자본+비유동부채) * 100 : 일반적으로 70% 수준
- 수익성 비율 : 수익성 판단 기준
  - 매출액 영업이익률 = 영업이익/ 매출액 * 100
  - 매출액 순 이익률 = 순이익 /매출액 * 100
  - 총자본 순 이익률 = 순이익 /총자본 * 100
- 활동성 비율 : 소유 자산이 얼마나 효과적으로 이용되고 있는지 측정하는 비율율
- 성장성 비율 : 기업이 적정하게 성장하는지 
  - 매출액 증가율
  - 총자본 증가율
- 생산성 비율 : 경영합리화 나타내는 비율
  - 부가가치율
  - 설비투자효율

### 지수법에 의한 거래처 재무평가 방법
평가항목 | 가중치 | 재무비율 | 산업평균 | 관계비율 | 조정 비율 | 평점
- 한국은행 홈페이지 경제통계 시스템, 동종업계 산업평균비율 조회
- 높을 수록 좋은 재무비율  (= 거래처 비율 / 동종업계평균비율)
- 낮을 수록 좋은 재무비율 (=동종업계평균비율 / 거래처 비율)
- 조정 관계비 = 관계비율 * 0.7
- 평점 = 가중치 * 조정관계비율 (평점이 70보다 크면 양호)

### 신용정보회사의 신용정보를 활용하는 방법

### 채무이행 심리를 활용한 채권회수기법
- 채권회수 마인드
  - 채권회수 완료해야 회사에 이익
  - 부실채권 역승수효과
- 채권은 1~2개월 내에 회수하도록 해야 함 (시간이 지날수록 채무이행의지 희박해짐)
- 변제 약정일 (1차 약정일 회수 안되면 2차 약정일 15일 이내 다시 약속) : 3~5차례 연기 전까지 회수 가능해짐
- 면담
  - 먼저 변제청구
  - 다툼 X
  - Win-Win하는 협상
  - 끈기있고 집요하게
  - 논리적으로 설득
  - 지불 계획서(각서)를 가지고 만남
- 유의할 점
  - 통화시간은 짧게
  - 감정을 상하게 하지 않음
- 채무자 유형별 채권회수 방법
  - 연체기간 짧고 지불의사 있는 경우 : 변제 약정일 약속 받고 회수활동, 연장회수는 3~5회 넘기지 않게함
  - 연체기간 짭고 지불의사 없는 경우 : 독촉장 , 법 조치
  - 연체기간 길고 지불 유예 요청 : 지불각서, 공정증서받고 어길 경우 법 조치 
  - 연체기간 길고 지불의사 없는 경우 : 법 조치
- 채권회수 절차
변제 기일에 변제 청구 -> 다음 변제 기일에 변제 청고 -> 간접강제(수시방문 독촉, 연대보증인 청구) -> 법 조치 -> 소제기 -> 강제 집행
- 채권회수 대상과 재산
  - 채무자가 개인 : 채무자, 보증인, 채무인수인
  - 일상가사대리(자녀학비..)는 부부연대채무
  - 채무자가 법인 : 채무자 법인, 보증인

### work sheet 활용법
채권회수를 체계적으로 하기위해 사용
- 회수계획은 변제약정일 기준으로 수립
- 변제약정일은 구두 또는 계약서에 명시
- 한달 반이 지나가면 전화가 아닌 방문

### 간접강제의 활용방법
- 일반적 간접강제
  - 악성채무자는 통화시간 관계없이 끈질길게
  - 악성채무자는 통화보다 방문(일부 회수 또는 각서받기)
  - 연대보증인이나 물상보증인에게 변제 청구
  - 최고장(독촉장)에 일정한 기간(7~15일)주어 채무이행하도록 함, 기간 내 이행하지 않으면 법 조치
  - 집행권훤 획득하면 재산명시 신청과 채무불이행자 명부등재 신청(돈이 있는데 갚지 않는 경우)
  - 채권자로서 채무자의 파산신청
